TextCraft is a powerful, modern, and user-friendly text analysis tool built with React. It offers a suite of features to manipulate and analyze text efficiently, wrapped in a stunning, responsive interface with dark mode support.

## üöÄ Features

- **Text Analysis**: Real-time word count, character count, and estimated reading time.
- **Text Manipulation**:
  - Convert to Uppercase
  - Convert to Lowercase
  - Remove Extra Spaces
  - Clear Text
  - Copy to Clipboard
- **Modern UI/UX**:
  - Glassmorphism design with beautiful gradients
  - Smooth animations and transitions
  - Fully responsive layout for all devices
  - Custom alerts with progress bars
- **Dark Mode**: Seamless toggle between light and dark themes for comfortable viewing.
- **Accessibility**: Designed with accessibility in mind.

## üì¶ Installation & Setup

Follow these steps to get the project running locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/choudhari-pranav/TextCraft_V1.git
    cd TextCraft_V1/TextCraft
    ```

2.  **Install dependencies:**
    Make sure you have Node.js installed. Then run:
    ```bash
    npm install
    ```
    This will install all required packages including `react`, `react-dom`, `react-router-dom`, `bootstrap`, and `react-icons`.

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in browser:**
    Visit `http://localhost:5173` (or the URL shown in your terminal) to view the app.
